## 🚀 Getting Started (5 Steps)

### Step 1: Download
1. Visit the GBS Update Portal
2. Login with your credentials
3. Click "Download EXE"
4. Choose: Full Version or Trial Version

### Step 2: Install
1. Run the installer
2. Follow on-screen prompts
3. Choose installation location
4. Wait for completion (1-2 minutes)

### Step 3: Launch
1. Start the GBS application
2. Create your admin account
3. Enter company information
4. Choose backup location

### Step 4: Configure
1. Add product categories
2. Set up company details
3. Create user accounts for team
4. Configure GST rates

### Step 5: Start Using
1. Add products
2. Create your first invoice
3. Generate quotations
4. Explore reports

## 🔒 Security Features

### Your Data is Protected
- **Local Storage**: Data never leaves your computer
- **Encrypted Backups**: Backup files are encrypted
- **User Audit Logs**: Track who did what and when
- **Session Security**: Auto-logout and session management
- **Password Hashing**: Industry-standard SHA-256

### Best Practices
1. Use strong passwords (12+ characters)
2. Create daily backups
3. Regular backup copies to external drive
4. Keep software updated
5. Restrict user access by role

## ❓ FAQ

### Installation & Startup
**Q: How long does installation take?**  
A: Usually 1-2 minutes for a typical computer.

**Q: Do I need internet to use GBS?**  
A: No, GBS works completely offline. No internet required.

**Q: Can I install on multiple computers?**  
A: Yes! Each purchase includes installation on your licensed computers.

### Data & Backup
**Q: What if I accidentally delete an invoice?**  
A: You can restore from backup or contact support.

**Q: How do I backup my data?**  
A: Automatic daily backups are created. You can also manually backup anytime.

**Q: Can I move my database to another computer?**  
A: Yes! Use backup/restore or contact support for data migration.

### Features
**Q: What units can I use?**  
A: ANY unit! Type "Per Pair", "Bundle", "Kit", or any custom unit.

**Q: Can multiple people use GBS at once?**  
A: Each person needs their own computer. GBS is per-computer license.

**Q: Does GBS support multiple currencies?**  
A: v3.0 uses INR. Multi-currency coming in v3.1.

### Trial & Upgrade
**Q: How long is the trial?**  
A: 30 days from first launch.

**Q: What limits does trial have?**  
A: 100 invoices/quotations max, 3 users, community support.

**Q: How do I upgrade to full version?**  
A: Enter your license key in Settings → License menu.
