# GBS v3.0.0 - Release Notes

## Version 3.0.0 - March 2026

### 🆕 Major New Feature: Customizable Unit Fields

The most requested feature is now here! In v3.0, we've transformed the unit selection from a restrictive dropdown to a powerful text field.

#### What Changed?
- **Before (v2.x)**: Users had to select from a fixed list: Nos, Unit, MM, CM, M, Gram, Kg, Litre, ML, Pack, Box
- **After (v3.0)**: Users can now type ANY custom unit they need

#### Where It's Used
1. **Product Purchase Orders** - Specify receiving units
2. **Customer Invoices** - Bill customers with custom units  
3. **Quotations** - Quote with flexible measurements

#### Examples of Custom Units Now Supported
- Per Pair
- Per Set
- Per Bundle
- Per Kit
- Per Carton
- Custom weight measurements
- Custom volume measurements
- Any business-specific unit

### 💡 Why This Matters
- **Flexibility**: No more workarounds for non-standard units
- **Accuracy**: Measure products exactly as you sell them
- **Simplicity**: Type once, use everywhere
- **Business Growth**: Support new product types and services

## Migration Guide

### Upgrading from v2.x to v3.0

1. **Download** the v3.0.0 installer from the update portal
2. **Close** GBS application
3. **Run** the installer - it will automatically backup your data
4. **Wait** for installation to complete
5. **Launch** GBS v3.0 - your data is preserved!

### Data Backup
- Automatic backup created before upgrade
- Data is 100% compatible between versions
- No data migration needed
