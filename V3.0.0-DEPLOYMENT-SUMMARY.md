## 🎯 Major Feature: Customizable Unit Fields

### What Changed
**v2.x (Old)**
- Unit selection via dropdown
- Limited to: Nos, Unit, MM, CM, M, Gram, Kg, Litre, ML, Pack, Box
- Restricts business flexibility

**v3.0 (NEW)**
- Unit field is now a free-text input
- Users can type ANY unit they need
- Examples: "Per Pair", "Per Set", "Per Bundle", "Per Kit", "Per Carton"

### Implementation
- **Files Modified**:
  - `Data-Sync-Tool/GstBilling.Desktop/MainWindow.xaml` (Lines 1260-1263 & 1639-1640)
  - Changed 2 ComboBox controls to TextBox controls
  
- **Binding Properties**:
  - `ProductQuantityUnit` (Purchase orders)
  - `QuotationItemUnit` (Quotations)

- **Backward Compatibility**: ✅ Fully compatible - existing data is preserved

## 📱 Installation Instructions (For Portal Users)

### For Professional Edition Users
1. Click **"Download EXE"** button on portal
2. Save **GBS-Setup-3.0.0.exe** to your computer
3. Run the installer
4. Choose installation location (default: C:\Program Files\GBS)
5. Wait 1-2 minutes for installation
6. Click "Finish" and "Launch GBS"
7. Enter your credentials and start using v3.0.0!

### For Trial Users
1. Click **"Download Trial"** or **"Download EXE"** for trial version
2. Save **GBS-Trial-Setup-3.0.0-trial.exe**
3. Run installer (same process as above)
4. 30-day trial automatically activated
5. All features available during trial period
6. Upgrade to professional version to continue beyond 30 days
